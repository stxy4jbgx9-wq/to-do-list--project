📝 To-Do List (Python CLI App)
A simple command-line To-Do List application built with Python.
This program allows users to manage tasks by viewing, editing, and deleting them through a menu-based interface.
This project is designed as a beginner Python practice project to understand lists, loops, and user input.

🚀 Features
Add multiple tasks at once using comma-separated input
View tasks in a numbered list
Edit an existing task
Delete a task
Interactive menu system
Simple command-line interface

🛠️ Technologies Used
Python
Command Line Interface (CLI)

📚 Concepts Practiced
Lists
Loops (while, for)
Conditional statements (if, elif, else)
User input handling
List operations (append, pop, indexing)
